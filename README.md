# Aerorewards 🏆

Aerorewards is a website to redeem points; choosing from a variety of interesting products related
to technology. Application based on Aerolabs design and concept.

### Technologies used 👨🏼‍💻

- React ⚛️
- React Router
- JavaScript
- Styled-Components 💅🏼
- Figma
